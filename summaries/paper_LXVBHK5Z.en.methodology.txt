The research design adopts an agentic LLM framework to automate multilingual grammatical analysis by reasoning over annotated corpora. The system integrates natural-language task interpretation, code generation, and data-driven reasoning to produce interpretable answers grounded in linguistic data. The framework is tested on Universal Dependencies (UD) corpora, focusing on word-order features inspired by the World Atlas of Language Structures (WALS). The evaluation covers 13 word-order features across over 170 languages, assessing performance through three dimensions: dominant-order accuracy, order-coverage completeness, and distributional fidelity.

The model/algorithm choices center on an agentic LLM architecture that decomposes tasks into subproblems, leveraging code generation for precise data extraction and analysis. The system interprets user queries in natural language, translates them into executable code (e.g., Python scripts) to query the UD corpora, and synthesizes results into interpretable reports. This approach balances flexibility in handling diverse linguistic queries with the rigor of structured data analysis.

Baselines for comparison include traditional rule-based methods and static LLM prompting without agentic capabilities. Rule-based systems lack adaptability to nuanced linguistic variations, while static LLMs may struggle with complex, multi-step reasoning tasks. The agentic framework is hypothesized to outperform these baselines by dynamically adapting to corpus-specific structures and providing transparent, data-grounded outputs.

The evaluation strategy employs three complementary metrics to assess system performance. Dominant-order accuracy measures the correctness of identified primary word-order patterns. Order-coverage completeness evaluates the system’s ability to detect all relevant word-order variations within a language. Distributional fidelity quantifies how accurately the system captures the frequency and distribution of these patterns. These dimensions collectively ensure a holistic assessment of the system’s generalization, comprehensiveness, and precision in grammatical analysis.

Assumptions include the availability of high-quality UD corpora with consistent annotations and the LLM’s ability to accurately interpret linguistic terminology. The framework assumes that word-order features can be reliably extracted from UD trees and that the agentic LLM can effectively bridge the gap between natural-language queries and structured data analysis. Limitations may arise from noise in corpus annotations or the LLM’s occasional misinterpretation of complex linguistic constructs.
